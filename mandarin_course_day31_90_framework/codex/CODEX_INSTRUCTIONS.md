Build Days 31–90 of the Traditional Mandarin 90-day intensive course using the provided framework.

Do not create light app-style lessons. Each day is a complete intensive learning unit.

The learner has completed Days 1–30 and has a closed vocabulary, grammar, sentence, pronunciation, listening, and writing database. Days 31–90 must extend that database without violating closed-vocabulary dependency rules.

Every sentence, dialogue, exercise, grammar example, listening script, speaking prompt, and audio item must use only:
1. vocabulary introduced on prior days, or
2. vocabulary introduced in the same day.

No future vocabulary may appear.

Every grammar point must include:
- function
- pattern
- meaning
- real usage
- when to use
- when not to use
- word order logic
- contrast with prior grammar
- English contrast
- Korean contrast
- Taiwan usage notes
- correct examples
- incorrect examples
- transformation drills
- open production drills
- error feedback map

Days 31–45 focus on survival fluency expansion.
Days 46–60 focus on narration and social conversation.
Days 61–75 focus on real-world transactions and problem solving.
Days 76–90 focus on independent daily communication.

The final Day 90 target is not native fluency. It is practical, smooth everyday communication in Taiwan across common non-technical domains.

Implement validators before rendering lessons.
If a lesson contains unknown tokens, missing grammar IDs, missing audio IDs, or future vocabulary, fail the build.
